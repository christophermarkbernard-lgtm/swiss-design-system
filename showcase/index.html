11 — Principles

Seven Rules

The constraints that define the system.

01

No Gradients

Flat color only. Gradients introduce visual noise that competes with typographic hierarchy.

02

No Shadows

Borders and whitespace create depth. Shadows are decorative — borders are structural.

03

0–2px Radius

Geometric precision. Rounded corners soften meaning; Swiss type is assertive.

04

No Decorative Icons

Icons must carry meaning. When in doubt, use words.

05

Typography Leads

Size and weight contrast communicates hierarchy before color does.

06

Grid Always

Every element aligns to the 4px base grid. No arbitrary values.
